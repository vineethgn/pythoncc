Lexical analysis

The first stage of the pythoncc compiler is the lexical analysis. It is done by a scanner which generates tokens for the next stage. The scanner or tokenizer is generated by making use of the flex scanner generator.

The rules for the definition of regular expressions of the tokens are taken from the Python programming language documentation.
http://docs.python.org/dev/reference/index.html
