Conclusion

The software project cycle was completed with a partial output. All the stages of the compiler were implemented. However, the desired features of the programming language were not.

The project gave the opportunity to explore the various techniques and conventions used in modern compiler design. Also familiarized with the compiler development tools such as flex, Bison & Memphis; code generation API LLVM and internals of the GCC compiler suite.


Fayaz Yusuf Khan 2010-03-25
