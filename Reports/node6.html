About the project

The aim of the project is to create a fully compliant Python compiler for the Linux platform. The creation of a compiler for a very high level interpreted language like Python may lead to the acceleration of application program development cycles. The final product of the project is a program called 'pythoncc'. This program will give native Linux executable code as output when a Python script is given as input. This will hopefully remove the need of the Python interpreter to be present in the memory every time of execution. It will also lead to better performance, that may be comparable with the existing compiled programming languages like C/C++, OCaml, etc.
