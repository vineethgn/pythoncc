INDENT and DEDENT token pairing

All the INDENT and DEDENT tokens generated must match. Also, the tokens must appear at the correct place, ie. the INDENT token must be generated at the beginning of an indented code block, and the DEDENT token must be generated at the end of the code block (beginning of the next line).

Multiple DEDENT tokens may be generated at the same time, but not INDENT. Also, an INDENT token cannot appear immediately after a DEDENT.
