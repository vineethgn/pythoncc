Abstract syntax tree

Figure 5.1: The AST class hierarchy UML diagram

The tree structure used is roughly a binary tree representation. Each node in the tree is an object a custom defined C++ class called Node, or one of its derivatives.

The lower level details of what exactly happens when a specialized node is created is hidden from the parser stage.

The abstract syntax tree may be generated using Memphis, [11] but currently, it is done manually.
