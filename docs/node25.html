Syntactic analysis

The next stage of the compiler is the syntax analysis or the parse-tree generation. It is done by an LALR parser, which is generated from Bison.

The Bison program takes the context-free grammar defining the language as input and generates the parser. [6] This parser will be able to accept tokens as input and generate parse-trees or abstract syntax trees (AST).

The grammar for Bison is based on the Python language reference manuals.
http://docs.python.org/dev/reference/index.html
