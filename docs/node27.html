Lowering of AST to LLVM IR

The AST itself contains methods for the code to be lowered to the next phase. The driver function only calls the root->generate(); method. This method causes the tree to be walked in an in-order traversal manner. Each node emits its IR code when it is traversed. This is done efficiently by making use of the polymorphic feature of the C++ object methods.
